# Pandas

All Pandas work uses the Titanic dataset. The goal wasn't just to learn the syntax — it was to answer real questions about the data and build a feature matrix ready for ML.

| File | What it covers |
|---|---|
| `series_df.py` | Series vs DataFrame, single vs double bracket, iloc vs loc |
| `importing.py` | read_csv, head, info, describe, shape |
| `titanic_eda.py` | Full EDA: null handling, groupby, filtering, 10 analytical questions |

`titanic_eda.py` is the main file. It produces the findings in the top-level README and ends with a cleaned DataFrame that feeds directly into the feature engineering step.
